### T2I vs I2V 선택 기준

- T2I(Text→Image): 장면 구성과 스타일을 처음 설계할 때 사용 (문장으로 장면을 정의하고, 최고의 한 컷을 뽑기 좋음).
- I2V(Image→Video): 확정된 키 비주얼에 자연스러운 줌인/모션을 추가할 때 사용 (일관된 스타일을 유지하면서 짧은 영상으로 확장하기 좋음).
